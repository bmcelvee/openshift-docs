= Adding RHEL compute machines to an {product-title} cluster
include::modules/common-attributes.adoc[]
:context: adding-rhel-compute

toc::[]

In {product-title}, you can add Red Hat Enterprise Linux (RHEL) compute, or
worker, machines to a user-provisioned infrastructure cluster. You can use RHEL
as the operating system on only compute machines.

[NOTE]
====
Before proceeding to add RHEL compute machines to an {product-title} cluster, check the required ports and xref:../installing/install_config/configuring-firewall.adoc#configuring-firewall[configure your firewall].

You can find the required ports information in the installing guide for your user-provisioned infrastructure platform.
====
