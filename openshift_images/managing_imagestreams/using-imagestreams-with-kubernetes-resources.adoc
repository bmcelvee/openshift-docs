= Using imagestreams with Kubernetes resources
include::modules/common-attributes.adoc[]
:context: using-imagestreams-with-kubernetes-resources
toc::[]

Imagestreams, being {product-title} native resources, work out of the box with
all the rest of native resources available in {product-title}, such as builds or
deployments. It is also possible to make them work with native
Kubernetes resources, such as jobs, replication controllers, replica sets or
Kubernetes deployments.
